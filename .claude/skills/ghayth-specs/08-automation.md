# 4. خريطة الترابط

| **العملية** | **يقرأ ثم يُنشئ ثم يؤثر** |
| --- | --- |
| بلاغ صيانة | عميل + فنيين + SLA ← مهمة + إشعارات ← KPI + قيد + ملف العميل |
| تسجيل حضور | وردية + سياسة ← حضور + مخالفة + جزاء ← الراتب + الصفحة الرئيسية |
| طلب إجازة | رصيد + جدول ← طلب + موافقة ← رصيد + حضور إجازة في كل التعيينات + إعادة توزيع مهام |
| فاتورة | عميل + بنود ← قيد + مهمة تحصيل ← ميزانية + رصيد عميل + بوابة العميل |
| رحلة أسطول | مركبات + سائقين ← مهمة + GPS ← تكاليف + KPI |
| فرصة CRM | عميل ← متابعات مجدولة ← won: عقد + فاتورة │ lost: تحليل ← ملف العميل |
