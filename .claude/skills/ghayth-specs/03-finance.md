# 2. سيناريو فاتورة مبيعات — 7 خطوات

فاتورة لأحمد المطيري 10 وحدات × 1000 = 10000 + VAT 1500 = 11500 ريال:

| **#** | **العملية** | **التفاصيل** |
| --- | --- | --- |
| 1 | حفظ الفاتورة INV-089 | INSERT INTO invoices مع clientId 42 وtotal 11500 وstatus draft |
| 2 | ← إرسال للعميل | status sent ثم إيميل PDF وواتساب رابط دفع. العميل يرى في بوابته (ملف 1) |
| 3 | ← ← قيد محاسبي تلقائي | مدين ذمم مدينة 11500. دائن إيراد 10000 + ضريبة 1500. يتحقق التوازن |
| 4 | ← ← ← رصيد العميل | أحمد من صفر إلى 11500 مدين. يظهر في ملفه 360 (ملف 1) |
| 5 | ← ← ← ← تحديث الميزانية | إيرادات + 10000 |
| 6 | ← ← ← ← ← جدولة تحصيل | INSERT INTO tasks تحصيل بعد 30 يوم في تقويم المحصّل. Haversine 12.3 كم و20 دقيقة |
| 7 | ← ← ← ← ← ← ربط بالتعيين | كل سجل يحمل assignmentId إذا الموظف عنده أكثر من تعيين |
