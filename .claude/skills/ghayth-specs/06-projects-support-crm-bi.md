# القسم 1: المشاريع — سيناريو 7 خطوات

مشروع تطوير تطبيق 150000 ريال 3 أشهر: إنشاء مراحل ومهام وتبعيات ← Critical Path: تصميم UI ثم backend ثم اختبار = 55 يوم ← المهام في تقويم كل عضو ← إكمال مهمة يفتح المهام المعتمدة عليها ← تأخر يحسب الأثر المالي ← إكمال مرحلة = فاتورة تلقائية للعميل ← تحذير ميزانية 80 بالمئة.

# القسم 2: الدعم — سيناريو تذكرة 8 خطوات

عميل يفتح تذكرة فاتورة خاطئة: أولوية تلقائية من الكلمات ← SLA رد 1 ساعة وحل 4 ← تعيين Load Balancing ← مهمة في الصفحة الرئيسية ← رد خلال 30 دقيقة SLA الرد تمام ← بعد 3 ساعات تحذير SLA ← إذا يحتاج زيارة تتحول لمهمة ميدانية بمسافة ← إغلاق + استبيان بعد 24 ساعة.

# القسم 3: CRM — سيناريو فرصة بيع 7 مراحل

فرصة 80000 ريال: حفظ prospecting ← جدولة متابعة 3 أيام ← كل مرحلة تجدول المتابعة التالية تلقائياً ← لم يتابع: تذكير ثم تصعيد لمدير المبيعات ← closed_won: عقد + فاتورة + KPI ← closed_lost: سبب إجباري + تحليل.

# القسم 4: التواصل

بريد وارد وصادر وخطابات بترويسة ومعاملات بتتبع وإعلانات.

# القسم 5: BI والتقارير

Dashboards وتقارير وKPIs ومحرك قرارات. جزئي — يحتاج: ديناميكي وReport Builder ومحرك حقيقي.
