# الفصل 4: الذكاء الاصطناعي — 6 أدوار

| **الدور** | **التفاصيل** |
| --- | --- |
| استقبال الطلب Front Desk AI | يفهم السلام معي 4 أشخاص نبغى عمرة من جدة بالطيارة ← يحلل 8 عناصر ← ينشئ ملف طلب ← يحوّل لموظف المبيعات |
| اقتراح الردود | مسودة رد كاملة تناسب نوع العميل واللغة والخدمة. مع الوقت يتعلم من ردود الموظفين المميزة |
| الترجمة الفورية | عميل بالإندونيسية: يترجم للموظف بالعربي ثم يترجم رد الموظف للإندونيسية. عربي وإنجليزي وإندونيسي وأوردو |
| تلخيص المكالمات | بعد مكالمة 15 دقيقة: ملخص + كشف الالتزامات. سلطان وعد يرد خلال يومين ← مهمة تلقائية |
| محرك القواعد Rules Engine | شكوى ← خدمة العملاء + إشعار المدير. عمرة رمضان ← أسعار الموسم. رقم جديد ← فتح ملف عميل. VIP ← المدير مباشرة |
| التنبؤ والذكاء التسويقي | توقع ضغط رمضان + اقتراح عروض + كشف عملاء مهددين بالانسحاب |

# الفصل 5: واجهات الموظفين

لوحة ويب: قائمة المحادثات في اليسار والمحادثة المفتوحة في الوسط وملف العميل في اليمين واقتراح AI في الأسفل.

تطبيق موبايل: واتساب الشركة + المكالمات + الإشعارات. الاتصال الصادر يظهر للعميل الرقم الموحد.

# الفصل 6: لوحة الإدارة

أرقام لحظية: 184 محادثة اليوم ومتوسط رد 2 دقيقة ورضا 4.3 وأكثر قسم العمرة 60 بالمئة.

ملخص AI يومي يُرسل واتساب للمدير: استقبلنا 184 استفسار و60 بالمئة عمرة رمضان و12 شكوى واقتراح زيادة الموظفين الأسبوع القادم.

# الفصل 7: سيناريوهات كاملة

حجز عمرة 10 خطوات: واتساب ← AI يرد ← سلطان يرسل العروض ← العميل يختار ← رابط دفع ← فاتورة تلقائية ← تأكيد حجز ← تذكير قبل السفر ← بعد العودة تقييم.

شكوى عميل 7 خطوات: الباص ما جا ← AI يكتشف شكوى طوارئ ← تصعيد فوري ← رد سريع ← تذكرة SLA ← متابعة ← تحليل.
