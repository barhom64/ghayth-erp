# لقطة حالة نظام غيث

> **تاريخ اللقطة: 18 يونيو 2026 — محدَّثة جزئيًا 21–22 يونيو 2026** (تحقّق فجوات السلامة المالية، اكتمال درج الإنشاء، توحيد الإدخال المالي، مسح المسارات 9–14، إغلاق بنود معمارية: RBAC حوكمة + توحيد مساري CRM + بُعدا fleetEngine وجرد دورة العد؛ FX revaluation الخيار أ مُنفَّذ #2873 + إصلاح خلل idempotency #2897 بتحقّق تشغيلي؛ لوحة الصدق مُكمَّلة: قياس القيود اليتيمة #2874 + بوابة إقفال حاجبة B3 #2902؛ FX مكتمل ومتماسك: تنظيف legacy #2907 + إحياء الطابور المؤجَّل بحارس ازدواج صلب #2924 + إصلاح تعدّد العملات #2927؛ توحيد preflight #2909؛ مسح آمن #2917). هذه نقطة زمنية ثابتة من ذاكرة إدارة المشروع، **لا تتحدث تلقائيًا.** عند أي تعارض، الكود الفعلي وحالة الـ PRs الحية هي الحقيقة. حدّث هذا الملف يدويًا بعد كل تغيير جوهري.

## البنية التقنية الجارية (تطوّرت عن المواصفات الأصلية)

pnpm monorepo: `api-server` (Node.js/Express + PostgreSQL + Drizzle ORM) — `ghayth-erp` (React 19 + TypeScript + Tailwind) — `ghayth-erp-deck`. الهجرات (migrations) ملفات مرقّمة في `src/migrations`. النطاق: 3500+ commit، 3500+ ملف، 14 نطاقًا وظيفيًا، 300+ هجرة، 600+ صفحة، 100+ ملف مسار backend، 1200+ نقطة API.

- **فهرسة الحقول المشفّرة:** لا يمكن فهرسة الحقول المشفّرة مباشرة (AES-GCM IVs عشوائية). استخدم نمط Encrypted+Hash: AES-256-GCM للتخزين، HMAC-SHA256 للبحث.
- **Mock-First للتكاملات الخارجية:** 3 شروط مسبقة قبل بناء المحوّلات الحقيقية.
- **ProductSelect مشترك واحد** يُبنى مرة (المستودعات) ويُعاد استخدامه — لا تكرار.
