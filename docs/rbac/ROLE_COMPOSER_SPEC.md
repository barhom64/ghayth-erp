## 3. قواعد

1. **يقرأ الميزات من `feature_catalog`** (المرآة) — لا تغيير كود لإضافة ميزة.
2. **تجميع بالمسار** (module) ثم الميزة ثم الإجراء — يطابق الصلاحية الخماسية.
3. **ملخص عربي دائم** أسفل الشاشة (مبدأ #1413: سهل لغير التقني).
4. **سجل التغيير** في `rbac_role_history` (before/after + سبب).

## 4. القرارات

- **يُطوَّر `/admin/roles` الموجود** نحو هذا التصميم؛ **يُدمَج** مع `/admin/rbac-matrix` (واجهة واحدة) — لا واجهة ثالثة.
- **توحيد الكتالوج** (RBAC-003): الواجهة تقرأ `feature_catalog` فقط؛ يُهجَر النموذج المسطّح بعد الاشتقاق.
- **القوالب الجاهزة** (#1413 §5) تُزرَع كـ `is_template` roles وتُعرَض في الإنشاء السريع.
