## معايير القبول التقنية الدائمة (لكل PR مالي)
1. guard أخضر بلا override (typecheck + 10k+ وحدة + كل التدقيقات).
2. أي تغيير ترحيل: اختبار تكامل حي يؤكد DR=CR + الحسابات عبر المُحلّل المركزي + الأبعاد + الفرع/الشركة + Audit (القاعدة 4).
3. المعاينة = القيد الفعلي (أي فرق = خلل حرج فئة 4).
4. الحالات الثلاث متسقة (لا مسودة مدفوعة/لا مرحّل بلا قيد).
5. idempotency لكل مسار كتابة (sourceKey ثابت؛ الإعادة لا تكرر أثرًا تجاريًا).
6. مرجعية اختبارات القبول AT-001..AT-020 (مصفوفة جرد المالك) — تُشغَّل من UI إلى JE على قاعدة حية عند الإغلاق (FIN-033/W9).
