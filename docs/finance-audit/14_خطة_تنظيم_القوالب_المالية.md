# 14 — خطة تنظيم القوالب المالية (بعد جرد 06)

## القواعد
- قالب واحد لكل غرض من الاثني عشر؛ الاستثناء بسبب تشغيلي مكتوب فقط.
- كل قالب يسحب من المستند + القيد المرحّل (لا جداول ظل)، عبر الطباعة الموحدة (`templateResolver` + branded themes) وأرشفة `print/archive`.
- التحقق: لكل قالب اختبار توليد يقارن أرقام الـPDF بقيمه في `journal_lines`/المستند.

## مخرَج
جدول: الغرض → القالب القائد → المكررات المرشحة للتقاعد → مصدر البيانات → الحالة.
