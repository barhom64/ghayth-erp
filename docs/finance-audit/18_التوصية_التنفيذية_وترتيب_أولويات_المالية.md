# 18 — التوصية التنفيذية وترتيب أولويات المالية

## خلاصة الحالة
الأساس المحاسبي صار صلبًا ومثبتًا حيًّا: محرك القيد بعقد مختبر، الحسابات بالاشتقاق المركزي (مع إصلاح كسور SOCPA في الفواتير والسداد والكيانات)، idempotency شامل في مسارات الكتابة الجديدة، وسلامة معاملة الاعتماد محمية بالعزل. المتبقي نوعان: **فجوات إنفاذ** (GRN، الحالات الثلاث) و**ديون تنظيم** (تكرارات، نيّات في الكود، قوائم/قوالب/صلاحيات بلا جرد موثق).

## تنبيهات حاكمة للمدقق
1. **لا PR يجمع موضوعين** — درس #2022 (نجح تقنيًا لكنه خالف «صغير مركز» اضطرارًا).
2. **الـdump ملك HR** — أي حاجة مالية فيه تُسلَّم كطلب لا كتعديل.
3. **57 تكرار أرقام الهجرات التاريخية تبقى مجمدة** (قرارك) — السقّاطة تمنع الجديد.
4. قرارات §5 محسومة المبدأ («النظام يفهم الحالة أولًا») — أي حالة تحتاج قرارًا أدق تُرفع لك ولا تُخمَّن.
