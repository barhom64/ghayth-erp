## الغرض

يوثّق هذا الملف كل عملية في النظام تُنشئ قيداً محاسبياً (journal entry) تلقائياً، مع تفاصيل الحسابات المدينة والدائنة ونمط مفتاح المصدر (sourceKey) للحماية من التكرار.

---

## 1. محرك القيود المحاسبية

### الدوال الأساسية

| الدالة | الملف | الغرض |
|--------|-------|-------|
| `createJournalEntry` | lib/businessHelpers.ts | إنشاء قيد يومية مباشر |
| `createGuardedJournalEntry` | lib/businessHelpers.ts | إنشاء قيد مع حماية — يُعلّم الكيان المصدر بـ `glStatus='failed'` عند الفشل |
| `getAccountCodeFromMapping` | lib/businessHelpers.ts | جلب كود الحساب من `accounting_mappings` مع قيمة افتراضية |

### آلية الحماية (Guard)

```
createGuardedJournalEntry({
  sourceType: "invoice",        // نوع المصدر
  sourceKey: "INV-2026-001",   // مفتاح فريد (idempotency)
  guardTable: "invoices",       // الجدول المراقب
  guardId: 123,                 // معرّف السجل
  ...journalData
})
```

عند الفشل: يُحدّث `glStatus = 'failed'` في الجدول المراقب + يُسجّل في `financial_posting_failures`.

## 2. خريطة القيود المحاسبية التلقائية

### 2.1 الفواتير (Invoices)

| العملية | sourceType | الملف | المدين (Debit) | الدائن (Credit) | Guard |
|---------|-----------|-------|----------------|-----------------|-------|
| ترحيل فاتورة مبيعات | `invoice` | finance-invoices.ts | ذمم مدينة (AR) | إيرادات + ضريبة مستحقة (VAT Payable) | invoices |
| إشعار دائن | `credit_memo` | finance-invoices.ts | مردودات المبيعات + ضريبة مستحقة | ذمم مدينة (AR) | credit_memos |
| تحصيل دفعة | `invoice_payment` | finance-invoices.ts | نقد/بنك | ذمم مدينة (AR) | invoice_payments |
| دفعة مقدمة عميل | `customer_advance` | finance-invoices.ts | نقد/بنك | إيرادات مؤجلة | customer_advances |

**مثال قيد فاتورة مبيعات:**
```
مدين: 1200 ذمم مدينة          10,000 ر.س
دائن: 4100 إيرادات مبيعات      8,696 ر.س
دائن: 2160 ضريبة مستحقة        1,304 ر.س
```

## 4. سجل فشل الترحيل

| العمود | الوصف |
|--------|-------|
| `financial_posting_failures.id` | معرّف الفشل |
| `sourceType` | نوع المصدر (invoice, goods_receipt, ...) |
| `sourceId` | معرّف السجل المصدر |
| `error` | رسالة الخطأ |
| `createdAt` | وقت الفشل |

**صفحة المراقبة:** `/admin/posting-failures` — تعرض كل حالات الفشل مع إمكانية إعادة المحاولة.
