# ٩٩ — أمر التنفيذ لكلود كود (مُلزِم، بلا مجال للتخمين)

> انسخ هذا الأمر كما هو إلى كلود كود. هو **عقد تنفيذ**، لا اقتراح.

---

## الأمر

أنت تنفّذ على نظام **غيث** (ERP حيّ، عزل مستأجرين). التزامك بحرفيّة هذه الحزمة شرط قبول. **ممنوع التخمين**؛ عند أي غموض، اسأل ولا تخترع.

### ١. اقرأ أولًا (قبل أي كود)
اقرأ كامل `docs/governance/system-doctrine-libraries/` (الملفات ٠٠→١٠). الملف ٠٠ حاكم. لكل مجال مكتبة موحّدة واحدة (٠٢) — استوردها، لا تنسخ منطقها.

### ٢. القواعد القاطعة (يُرفض أي PR يخالفها)
1. **لا كتابة بلا فرع محدّد ودور مخوّل.** السياق (`companyId/branchId/userId/createdAt`) يُحقَن من `req.scope`/`NOW()` — يُتجاهَل من `req.body` (٠٠§٢، ١٠).
2. **تاريخ الإنشاء آليّ ومنفصل عن تاريخ المستند** (٠٠§٣، ٠٤).
3. **الترقيم** من `numberingService` فقط (٠٣). **التواريخ** من `businessHelpers`/`date-utils` (٠٤). **الطباعة** من `print-client` (٠٥). **القراءة/الفلترة** من `buildScopedWhere` (١٠).
4. **الصلاحيات**: بوّابات القدرة عبر `scopeCan` المشتقّ من المنح؛ بوّابات الموقع/النطاق تُترَك (٠٦). الاعتماد بـ`approve`/`reject` لا `update` (٠٧).
5. **الإدخال**: فرز ذكي → كشف تدريجي → استحضار الكيان → توجيه محاسبي آليّ (٠١، ٠٨). لا نموذج طويل، لا اختيار حساب يدوي.
6. **التفعيل**: حدّ أدنى + قالب وظيفة يولّد المتطلّبات + توزيع على المالكين (٠٩). لا «عهدة إن وجدت».
7. **لا تكرار**: صفحة/نموذج/قالب/مكتبة مكرّرة تُدمَج لا تُنسَخ.

### ٣. ترتيب التنفيذ (صارم — لا قفز)
أ. **الحزمة الحاكمة معتمدة** (هذا الملف) = المرجع.
ب. **مسار HR** بالترتيب الموثّق في `docs/hr/`:
   - HR-REV-0 (جرد) ✅ — HR-REV-1 (الأدوار/الصلاحيات: الكود الخلفي ✅، يتبقّى اعتماد المصفوفة + تحقّق ظهور الشريط لكل دور).
   - ثم **HR-REV-2** (تنظيف الصفحات المكرّرة) — لا يبدأ قبل إغلاق HR-REV-1.
   - ثم **HR-REV-3/4** (التفعيل السريع + قوالب الوظائف، ٠٩) — أعلى قيمة تشغيلية.
   - ثم HR-REV-5/6/7 (نماذج/عرض/سياسات) — ثم HR-REV-8/9 (بوّابة القبول + المرفقات).
ج. **لا تفتح موجة تطوير قبل اعتماد جردها/مصفوفتها.**

### ٤. انضباط كل PR
- صغير ومركّز، رسالة ووصف **مطابقان للكود حرفيًّا**.
- `typecheck` + `guard` أخضر قبل الدمج. لا حذف صلاحيات/سلوك تشغيلي.
- لا خلط بين مهام HR ومهام أخرى في PR واحد.
- كل بوّابة قبول تشغيلية تُثبَت برحلة حيّة (HR-REV-8)، لا ادّعاء نظري.

### ٥. عند الشكّ
توقّف واسأل. تسليمك يُقاس بـ: صفر تكرار، صفر تخمين، صفر كتابة بلا سياق/دور، توحيد كامل للمكتبات، واعتماد آليّ منضبط.

— انتهى الأمر.
