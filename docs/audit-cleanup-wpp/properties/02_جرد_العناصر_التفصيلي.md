## 5) آلات الحالة

| الكيان | الحالات | الانتقالات النهائية |
|---|---|---|
| `property_units` | available / rented / maintenance / under_maintenance / out_of_service / reserved | لا نهائيات (كلها قابلة للعكس). **مصدر واحد بعد #2057** في `lifecycleEngine.STATE_MACHINES`. |

| `maintenance_requests` | pending → approved → assigned → in_progress → completed → closed (+ rejected/cancelled/returned) | rejected/cancelled/closed نهائية. |
