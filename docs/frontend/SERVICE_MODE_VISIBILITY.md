# ظهور وضع الخدمة — SERVICE_MODE_VISIBILITY

> المرحلة 4 — **Ghaith Operating Foundation** (#1418) · 2026-05-29 · فرع `claude/ghaith-foundation-audit-wdIUf`
> **الهدف:** الخدمة المساندة (الخادم العام) تظهر **داخل سياق** المسار القائد، لا كنظام كامل مستقل. يعالج MENU-003.
> **يُبنى على:** `PATH_LEADER_SERVICE_MATRIX` + `CORE_SERVICES_INVENTORY` + المكوّنات المشتركة (`entity-documents`, `entity-comments`, `linked-tasks`…).

---

## 1. المبدأ

الخوادم العامة (الوثائق، التعليقات، المهام، المراسلات، التدقيق، الطباعة، التقويم) **ليست أقسامًا مستقلة** في تجربة المستخدم غير المخوّل — تظهر كـ **تبويبات/أزرار داخل صفحة الكيان القائد**.

```
صفحة فاتورة (القائد: Finance)
 ├── تبويب: التفاصيل
 ├── تبويب: المرفقات   ← خدمة الوثائق (في السياق)
 ├── تبويب: التعليقات  ← خدمة التعليقات (في السياق)
 ├── تبويب: المهام     ← خدمة المهام (في السياق)
 ├── تبويب: الخط الزمني← خدمة التدقيق (في السياق)
 └── زر: طباعة/تصدير   ← خدمة الطباعة (في السياق)
```

---

## 2. متى يظهر الخادم كقسم مستقل؟

| الحالة | الظهور |
|---|---|
| مستخدم عادي داخل رحلة | الخادم **في السياق فقط** (تبويب/زر) |
| مسؤول مخصّص للخدمة (مسؤول وثائق، مسؤول مراسلات) | قسم مستقل (لأن الخدمة هي مساره القائد) |
| ميزة الخدمة غير مفعّلة للمنشأة | لا تظهر إطلاقًا (تفعيل — VIS-002) |

> القاعدة: الخادم قسم مستقل **فقط** لمن الخدمة مساره القائد؛ لغيره يظهر داخل السياق.

---

## 3. التصميم

- المكوّنات المشتركة (`EntityDocuments`, `EntityComments`, `LinkedTasks`, `EntityTimeline`, `PrintButton`) تُركَّب كتبويبات في صفحة الكيان عبر `entity-detail-page`/`detail-page-layout` الموجودة.
- القائمة الجانبية لا تُظهر "الوثائق/التعليقات" كقسم لغير مسؤول الوثائق (قاعدة `MENU_GOVERNANCE` 3).
- ظهور التبويب يحكمه: تفعيل الخدمة + صلاحية المستخدم على الكيان.

---

## 4. قواعد

1. **الخادم في السياق افتراضيًا** — قسم مستقل استثناء (لمسؤول الخدمة فقط).
2. **لا تكرار:** نفس المكوّن المشترك في كل سياق — لا نسخة لكل مسار.
3. **محكوم بالتفعيل + الصلاحية:** التبويب يظهر إن فُعّلت الخدمة وملك المستخدم صلاحيتها.

---

## 5. القرارات

- **يُستخدم نمط التبويبات في `entity-detail-page` الموجود** — لا أقسام مستقلة للخوادم في قائمة المستخدم العادي.
- **يربط** `VISIBILITY_ENGINE_SPEC` (التفعيل) و`MENU_GOVERNANCE` (قاعدة 3).
- **اختبار القبول:** `RBAC_FRONTEND_E2E_SCENARIOS` (الخدمة تظهر في السياق لا كنظام).
