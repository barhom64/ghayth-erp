# FINANCE_NAVIGATION_REDESIGN.md

إعادة تنظيم القائمة المالية حسب دورة العمل (workflow-first) بدل القائمة
الطويلة المتضخمة.

## المجموعات الـ13

1. **لوحة المالية** — dashboard، CFO cockpit، نظرة عامة.
2. **العمليات اليومية** — فاتورة/سند/مصروف سريع، اختصارات الإنشاء.
3. **الخزينة والبنوك** — الخزينة، التحويل بين الحسابات، التسوية البنكية،
   مراقبة البنوك، التدفق النقدي + التوقعات.
4. **العملاء والمقبوضات** — الفواتير، المقبوضات، سند قبض، دفعات مقدمة،
   كشف العميل، مخاطر العملاء، ملف العميل 360°.
5. **الموردون والمدفوعات** — أوامر/طلبات الشراء، الموردون، فواتير الموردين،
   المدفوعات، دفعة الدفع، عقود الموردين، كشف المورد.
6. **المصروفات والعهد** — المصروفات، اعتماد المصاريف، موزّع التكاليف،
   العهد، تقرير العهد.
7. **دفتر الأستاذ والقيود** — القيود، القيود اليدوية (بصلاحية)، القوالب،
   القيود الدورية، طابور الترحيل، عكس قيد.
8. **مراكز التكلفة والتحميل** — مراكز التكلفة، شجرة CC، قواعد التوجيه،
   تشخيص التغطية، سجل التجاوزات، نتائج التخصيص.
9. **الأصول والمخزون والتكاليف** — الأصول الثابتة، السجل التحليلي، الإهلاك،
   تقييم المخزون، CIP، تكاليف المشاريع، Cost Center P&L.
10. **الضرائب والامتثال** — نظام الضرائب، رموز الضريبة، WHT، تسوية VAT،
    جاهزية ZATCA، تقويم الإقرارات، ZATCA Reports Hub.
11. **الميزانيات والإقفالات** — الميزانية، الفترات المالية، إقفال الفترات
    (بصلاحية)، فحص قبل الإقفال، إقفال السنة.
12. **التقارير المالية** — بوابة تقارير واحدة (الميزانية العمومية، قائمة
    الدخل، التدفقات، ميزان المراجعة، التقارير المخصصة).
13. **إعدادات المالية** — في آخر القائمة: دليل الحسابات، التصنيفات،
    accountUsage، السياسات، الإعدادات العامة.

## القواعد

- كل صفحة في **مجموعة واحدة فقط** (لا تكرار).
- الإعدادات آخر القائمة؛ التقارير في بوابة واضحة.
- القيود اليدوية + الإقفالات تظهر **حسب الصلاحية** فقط (minRoleLevel / perm).
- لا فلاتر مكرّرة بين الصفحات — تُوحَّد عبر المكوّنات المشتركة.

## التنفيذ

التغيير في `navigation.registry.ts` فقط (مصدر الحقيقة للسايدبار + /services).
لا تُمسّ مسارات `financeRoutes.tsx` (الروابط ثابتة)، فقط تُعاد المجموعة
والترتيب. هذا يضمن عدم كسر أي deep-link قائم.

## معايير القبول المرتبطة

- القائمة المالية مرتبة حسب دورة العمل، غير متضخمة.
- كل صفحة في مجموعة واحدة واضحة.
- الإعدادات في الآخر، التقارير في بوابة، القيود/الإقفالات حسب الصلاحية.
