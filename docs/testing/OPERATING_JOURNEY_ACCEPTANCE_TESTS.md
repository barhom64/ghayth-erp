## صيغة الاختبار: المعطى / عندما / إذن (+ الأثر المرئي)

### اختبار ج1 — إنشاء شركة حتى التفعيل
- **عندما:** ينشئ المدير شركة (هيكل + شجرة حسابات + أدوار).
- **إذن:** الشركة `active`، الفروع/الحسابات/الأدوار مزروعة، حدث `company.created`.
- **الأثر المرئي:** الشركة تظهر في الإعدادات + سجل تدقيق الإنشاء.
- **حالة معروفة:** 🚨 لا واجهة تهيئة (حاجز) — يُتحقَّق بعد التنفيذ.

### اختبار ج2 — موظف حتى الدخول
- **عندما:** إنشاء سريع موظف + حساب + دور (`POST /admin/onboard`).
- **إذن:** موظف + مستخدم + دور بمعاملة ذرّية، دعوة/كلمة مرور.
- **الأثر:** الموظف يدخل ويرى مركز عمله المناسب + سجل `user.onboarded` **بالصفة**.

### اختبار ج3 — مصروف حتى الأثر المالي
- **عندما:** ينشئ محاسب مصروفًا ويُرسله؛ يعتمده المعتمد.
- **إذن:** قيد `DR مصروف / CR نقد-AP` مُرحَّل.
- **الأثر:** الحالة draft→approved→posted + القيد في GL + سجل تدقيق بالصفة + إشعار للمنشئ.

### اختبار ج4 — رحلة نقل حتى الإغلاق والتقرير
- **عندما:** بدء رحلة → تسجيل وقود → إغلاق.
- **إذن:** قيد 4 أسطر + قلب حالة المركبة/السائق.
- **الأثر:** تقرير تكلفة الرحلة + الحالة completed.
- **حالة معروفة:** ⚠️ يُتحقَّق من **عدم ازدواج عدّ الوقود**.

### اختبار ج5 — مجموعة عمرة حتى الفاتورة والتقرير
- **عندما:** موسم → مجموعة → إصدار فاتورة → دفعة.
- **إذن:** فاتورة GL-حاجزة بأبعاد (وكيل/موسم) + تخصيص دفعة + عمولة بعد الرواتب.
- **الأثر:** تقرير ربحية العمرة + الحالة posted→paid.

### اختبار ج6 — عقد عقاري حتى التحصيل
- **عندما:** إنشاء عقد → استحقاق دفعة → تحصيل.
- **إذن:** ترقيم العقد + حالة الوحدة→مؤجّرة + التزام تجديد + قيد `DR نقد / CR AR`.
- **الأثر:** دفتر المستأجر + الحالة active.

### اختبار ج7 — قضية حتى الجلسة والتنبيه
- **عندما:** قضية → جدولة جلسة → حكم.
- **إذن:** التزام + إشعار + (مهمة — يُتحقَّق من الربط) + قيد تسوية للحكم.
- **الأثر:** تقرير القضايا + الحالة scheduled→completed.

### اختبار ج8 — رواتب حتى الأثر المالي
- **عندما:** تشغيل رواتب.
- **إذن:** قيد ذرّي (الكل أو لا شيء) + حساب عمولة.
- **الأثر:** تقرير الرواتب + حالة posted + إشعار CFO/HR + سجلان (حدث + فعل).

## معيار النجاح العام
كل رحلة: (1) تكتمل من الواجهة، (2) تُنتج الأثر المتوقّع، (3) يظهر الأثر في حالة/تقرير/سجل، (4) يُسجَّل التنفيذ **بالصفة** (الدور). الفجوات المعروفة (ج1/ج4/ج7) تُعاد بعد التنفيذ.
