# فهرس مسار المالية — هرمية المفاهيم (B7)

> مسار المالية موزّع على **21 ملف `routes/finance-*.ts`** بقائمة مسطّحة بلا هرمية —
> فلا يُعرف الجوهر من الأداة. هذا الفهرس يصنّفها (قاموس المفاهيم §5). **توثيق فقط**؛
> لا يُعيد تنظيم الملفات. أي ملف مالية جديد يُعلن طبقته في رأسه.

## قاعدة
- الملف الجديد في المالية **يعلن طبقته** في رأسه (Core / Recognition / Tool / Report / Integration).
- ما يمسّ الدفتر يبقى محصورًا في مسار الترحيل المعتمد (حرّاس #2660) — هذا الفهرس تنظيمي لا يغيّر ذلك.
