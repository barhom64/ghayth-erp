# AI Guardian Setup

## الهدف
إضافة حارس تقني للنظام يقوم بـ:
- تحليل أخطاء الكود
- كشف نواقص النظام
- كشف التعارضات والمخاطر
- اقتراح إصلاحات عملية

## المرحلة الحالية
Read-only analysis داخل النظام عبر Anthropic Claude API.

## المتطلبات
- ANTHROPIC_API_KEY في Replit Secrets
- Node.js backend
- نقطة API داخلية للتحليل

## المخرجات
يرجع الحارس:
1. ملخص الحالة
2. الأخطاء الحرجة
3. النواقص الوظيفية
4. المخاطر المعمارية
5. الإصلاحات المقترحة
6. الأولوية

## الخطوات القادمة
- ربط endpoint داخلي
- إرسال logs / code / issue descriptions
- لاحقًا إضافة tools للفحص الآلي
