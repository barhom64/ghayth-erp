## 3.ب — التكرار الوظيفي (مراجعة أعمق)

> الجرد الأول رصد تكرار **التسمية** فقط (=0). لكن يوجد تكرار **وظيفي**: مسارات مختلفة تؤدي نفس الوظيفة أو تتشارك المحرّك — تأكّد بفحص مصدر بيانات كل صفحة.

| # | العنقود | الدليل | الحكم |
|---|---|---|---|
| 1 | **ملفات 360 ×3** — الجهة (`/finance/entity-360`) + المورد (`vendor-360-sheet`) + العميل (`customer-360-sheet`) | vendor-360 وcustomer-360 **يستدعيان نفس المحرّك** `/finance/entity-360` (عام بـ`entityType`) | 🔴 تكرار محرّك — الفرعيّان غلاف يضبط النوع. يُقترح: مدخل واحد بمُبدّل نوع، والفرعيّان اختصارات/بحث (قرار مالك) |
| 2 | **ربحية مراكز التكلفة ×3** — مقارنة (`cost-center-pnl`) + تفصيل مركز (`cost-center-drill-pnl`→`/cost-centers/:id/pnl`) + ترتيب (`cost-center-ranking`) | ثلاث صفحات على نفس `cost-centers` بزوايا مختلفة | 🟠 تداخل — يُقترح دمجها كتبويبات في صفحة واحدة |
| 3 | **توقعات النقد ×2** — `cash-flow-forecast` + `cash-13week` | الصفحتان **تتبادلان الاستدعاء** + تلمسان `cashflow`/`cash-position-calculator` | 🟠 تداخل — توقّعان للنقد؛ يُقترح توحيدهما بمدى قابل للاختيار |
| 4 | **تقييم المخزون ×2** — `inventory-costing` + `reports/inventory-valuation` | محرّك تكلفة مقابل تقرير | 🟡 تداخل مفهوم — يبقيان مع ربط واضح |
| 5 | **أدوات التسوية ×4** — `reconciliation-hub` + `bank-reconciliation` + `account-recon-workpaper` + `vendor-settlement-workbench` | أربع أدوات تسوية على 3 مجموعات | 🟡 تشتّت — يُقترح جمعها تحت «مركز التسويات» كتبويبات |
| 6 | **تقادم الذمم** — «تقادم الذمم الدائنة» (AP) مقابل «تقادم الذمم» (AR، مُسمّى عامًّا) | لبس تسمية لا تكرار محرّك | ✅ **أُصلح:** → «تقادم الذمم المدينة» |

> **اعتراف:** «0 تكرار» في الجرد الأول كان دقيقًا **للتسمية فقط**؛ التكرار الوظيفي أعلاه فاتني حتى نبّه إليه مالك النظام. #6 آمنة وطُبِّقت؛ #1–#5 دمجٌ وظيفي يحتاج قرار المالك (بلا حذف مسار).

### 4.ب — إدراج الخدمات المخفية الثماني في مكانها الطبيعي

```
اللوحات والإقفال      ← + «جداول الإطفاء» + «الإيراد المؤجل»   [دورة الاستحقاق قبل الإقفال]
الأصول والعهد         ← + «الأعمال الرأسمالية تحت التنفيذ (CIP)»
المشتريات والموردين   ← + «مطابقة الفاتورة 3-way»
الفواتير والسندات     ← + «ترحيل أقساط التأمين»  (ضمن المصروفات/المدفوع مقدمًا)
محرك التوجيه المحاسبي ← + عنقود «التشخيص المحاسبي»:
                           «مركز التصنيف المحاسبي» + «طابور فشل الحسابات الفرعية»
                           + «إصلاح الحسابات مغلوطة الأبوّة»
```
