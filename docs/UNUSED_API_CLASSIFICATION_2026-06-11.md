## منهجية التصنيف

| الرمز | المعنى | القرار المقترح |
|---|---|---|
| **أ** | مستهلكة فعلياً خارج الواجهة الرئيسية (جوال/سكربتات رحلات/طباعة) أو إيجابية كاذبة للماسح الساكن | لا إجراء — توثيق فقط |
| **ب** | خدمة ناقصة: الخلفية جاهزة والصلاحيات مطبقة، لكن لا واجهة/زر يصل إليها | بناء الواجهة أو قرار تأجيل صريح |
| **ج** | مكررة أو متروكة لصالح نقطة أخرى مستهلكة | مرشحة للإزالة بعد فترة إهلاك |
| **د** | داخلية/تشغيلية (backfill، تشخيص، اشتراكات) لا يُتوقع لها واجهة | لا إجراء |
