# 09 — جرد التكاملات

> القاعدة: النقل قائد تشغيلي؛ كل تكامل «من زاوية النقل فقط».

---

## 1. العمرة → النقل (العمرة تقود، النقل يستقبل)

| القدرة | الموضع | الحالة |
|---|---|---|
| مصادر مرتبطة بانتظار نقل | `GET /transport/integration/linked-sources` (مجموعات عمرة + عقود تأجير بلا حجوزات) | ✅ |
| توليد برنامج 3 مقاطع | `POST …/from-umrah-group/:groupId` — مطار→مكة، مكة→المدينة، المدينة→مطار؛ idempotent على (group, routeType)؛ passengerCount=mutamerCount؛ تواريخ من حدود الموسم (يعدّلها المشغّل) | ✅ |
| فاتح من نموذج الحجز | `UmrahGroupPicker` يملأ المجموعة/الفندق/العدد بلا إدخال يدوي (A-01) | ✅ — الإثبات الحيّ في E2E |
| تخطيط جماعي | `POST …/plan-bookings` — محرك الإسناد لكل حجز + dedupe عبر الدفعة + عتبة نقاط 60 + تصنيف النتائج (planned/needs_attention/no_candidate/no_line/skipped) | ✅ |

## 2. المالية (مستهلك الأثر — Candidate فقط)

| القدرة | الحالة |
|---|---|
| مرشّح الحمولة | ✅ `createCargoBillingCandidate` عند ready_for_invoice (kg, delivered) |
| مرشّح التأجير | ✅ `createRentalBillingCandidate` عند الإرجاع (quantity=أيام، الإيراد على مدى المدة، الزائد بندًا مقترحًا) |
| **مرشّح الركاب** | ❌ **غير موجود** — TA-GAP-01 (الملف 03 §T9) |
| materialize (جهة المحاسب) | يدعم cargo_manifest فقط؛ rental يُرفض برسالة عربية واضحة «غير مدعوم بعد للترحيل التلقائي» — **مؤجَّل عمدًا** حتى مرحلة الترحيل المالي (بأمر المالك: لا لمس لدفتر الأستاذ) |
| service lines + invoice-batches | ✅ موجودة لخط الفوترة التجميعي |
| **حدّ «لا قيد من شاشة النقل»** | ✅ مثبَّت باختبارات grep سالبة على endpoints التشغيل |

## 3. HR (مرجع موظفين فقط)
- `fleet_drivers.employeeId` + `resolveDriverFromScope` (من هوية الجلسة). بطاقة الموظف المرتبط في driver-detail. لا منطق HR داخل النقل. ✅ ضمن الحدود.

## 4. CRM (مصدر العميل)
- إجبار `customerId` على إنشاء الحجز والبوليصة والتأجير (refinements خادمية + واجهات read-only). ✅.

## 5. Telematics (خادم تتبع)
- CMSV6 webhook (HMAC) + مزامنة/heartbeat/retention + قاطع دارة. يتقاطع مع النقل في live-map وscorecard فقط. ✅ ناضج.

## 6. أحداث النقل (العقد البيني)
كتالوج `FREIGHT_EVENTS` — 30 مفتاحًا بمسميات عربية كاملة: دورة البوليصة (11) + dispatch (5) + حجز + حراس السعة/الأهلية (5) + الفوترة (5) + checkpoints + أحداث المركبة. المستمعون عبر eventBus. ✅ مصدر حقيقة واحد.

## 7. خلاصة فجوات التكامل
