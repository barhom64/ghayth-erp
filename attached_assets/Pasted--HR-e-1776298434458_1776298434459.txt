## القسم 6: الرواتب والخصومات

### 6.1 تشغيل كشف رواتب — سيناريو 12 بند

**المرجع:** ملف 02 — سيناريو الرواتب

**المُنفِّذ:** hr_manager أو finance_manager

| # | الخطوة | النتيجة المتوقعة | الحالة |
|---|--------|------------------|--------|
| 6.1.1 | فتح صفحة الرواتب واختيار الشهر (مثال: مارس) | القائمة تعرض الأشهر المتاحة | ⬜ |
| 6.1.2 | التأكد أن الفترة المالية مفتوحة | إذا مُقفلة: رسالة "لا يمكن تشغيل الرواتب في فترة مُقفلة: [اسم الفترة]" | ⬜ |
| 6.1.3 | الضغط على "تشغيل الرواتب" | النظام يحسب لكل موظف/تعيين: | ⬜ |
| | | **الإضافات:** الأساسي + بدل السكن + بدل النقل + عمل إضافي | |
| | | **الخصومات:** تأخر + غياب + جزاء + سلفة + GOSI 9.75% | |
| | | **الصافي = الإضافات - الخصومات** | |
| 6.1.4 | مثال حسابي | أساسي 8,000 + سكن 2,500 + نقل 500 + إضافي 500 - تأخر 92 - غياب 267 - جزاء 267 - سلفة 1,000 - GOSI 780 = **7,594 صافي** | ⬜ |
| 6.1.5 | الخصومات المعلّقة | كل `attendanceDeductions` بحالة `pending_payroll` تُحتسب وتتحول إلى `applied` | ⬜ |
| 6.1.6 | كشف الراتب يُحفظ | `payroll_runs` + `payroll_lines` + `payroll_deductions` | ⬜ |
| 6.1.7 | قيد محاسبي تلقائي | يُنشأ قيد: مدين رواتب + دائن بنك + GOSI | ⬜ |
| 6.1.8 | سجل تدقيق | من شغّل الرواتب + متى + لأي شهر | ⬜ |

**رسائل الخطأ المتوقعة:**

| الحالة | الرسالة المتوقعة | ما يُعتبر فشلًا |
|--------|------------------|-----------------|
| الفترة المالية مُقفلة | "لا يمكن تشغيل الرواتب في فترة مُقفلة: [اسم الفترة]" | الرواتب تمر أو رسالة عامة |
| الرواتب شُغِّلت مسبقًا لهذا الشهر | "تم تشغيل رواتب [الشهر] مسبقًا" | تشغيل مكرر بدون تحذير |

## ملاحظات التنفيذ

1. **المستودع:** كل شيء يُرفع على المستودع الرئيسي (main). لا مستودعات فرعية.
2. **آلية العمل:** أنت تعدّل وترفع على main. Claude Code يساعد في الكود. Replit للبيئة التشغيلية.
3. **كل خطأ يُوثَّق:** عند فشل أي سيناريو، يُسجَّل هنا مع:
   - رقم السيناريو
   - الخطأ الفعلي (screenshot أو نص الخطأ)
   - الملف المسؤول (backend أو frontend)
   - الإصلاح المطلوب
4. **لا ننتقل لمسار آخر** حتى يصير المجموع: 0 فشل + 0 لم يُختبر.
5. **الأولوية:** الأقسام 1-6 أولًا (التدفقات الأساسية)، ثم 7-11 (الخدمات والتوحيد والأمان والأتمتة).
